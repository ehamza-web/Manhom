تحليل فني لتدفق البيانات، حالات الاستخدام، والتكامل مع الأنظمة الخارجية

1. مخطط سياق النظام (Context Diagram)

يوضح هذا المخطط نظام "من هم" كمركز للعمليات وكيفية تواصله مع المستخدمين والمنصات الخارجية (Octopus, n8n, Neverbounce).

graph LR subgraph External_Systems [الأنظمة الخارجية] Octopus[منصة Octopus] n8n[أداة n8n الأتمتة] Neverbounce[خدمة التحقق Neverbounce] end User((المستخدم / الزائر)) -- طلب اشتراك --> System[**نظام من هم - نشرة موشور**] Admin((مدير النظام)) -- عرض وفلترة المشتركين --> System System -- التحقق من البريد --> Neverbounce System -- Create/Update Contact --> Octopus n8n -- التحقق اليومي 12 AM --> System n8n -- Unsubscribe = true --> Octopus style System fill:#eff6ff,stroke:#3b82f6,stroke-width:3px

2. مخطط حالات الاستخدام (Use Case Diagram)

graph TD Admin((مدير النظام)) --- UC1(عرض جدول المشتركين) User((المستخدم/الزائر)) --- UC2(الاشتراك في النشرة) n8n((أداة n8n)) --- UC3(إنهاء الاشتراك التلقائي) subgraph Operations [العمليات] UC1 UC2 UC3 end UC1 -.-> Search(البحث بالبريد) UC1 -.-> Filter(الفلترة بالتاريخ والنوع) UC2 -.-> Validation(التحقق من البيانات و Neverbounce) style Operations fill:#fff,stroke:#94a3b8

3. سير عمل الاشتراك (Subscription Process)

flowchart TD Start([بداية طلب الاشتراك]) --> Input[إدخال البيانات: الاسم، الشركة، المنصب، الهاتف، البريد] Input --> Terms{الموافقة على الشروط؟} Terms -- لا --> EndFail([توقف]) Terms -- نعم --> Validate{صيغة البريد و Neverbounce؟} Validate -- غير صحيح --> ErrorMsg[عرض رسالة خطأ: البريد غير صحيح] ErrorMsg --> Input Validate -- صحيح --> OctopusSync[استدعاء خدمة Octopus: Create/Update Contact] OctopusSync --> CheckSync{نجاح المزامنة؟} CheckSync -- فشل --> FailMsg[رسالة: فشل تقديم الطلب، يرجى المحاولة لاحقاً] CheckSync -- نجاح --> DB[حفظ البيانات في قاعدة بيانات من هم] DB --> SetDates[تحديد تاريخ البداية والنهاية +30 يوم] SetDates --> Success([تم الاشتراك بنجاح]) style OctopusSync fill:#fef3c7,stroke:#d97706 style CheckSync fill:#fef3c7,stroke:#d97706

4. عملية إنهاء الاشتراك التلقائي (n8n Automation)

flowchart TD Time([الساعة 12:00 AM يومياً]) --> Fetch[جلب المستخدمين الذين تاريخ نهايتهم = اليوم] Fetch --> Loop{لكل مستخدم مستحق؟} Loop -- نعم --> UpdateOctopus[تغيير Unsubscribe إلى True في Octopus] UpdateOctopus --> Loop Loop -- لا يوجد المزيد --> Finish([نهاية العملية]) style UpdateOctopus fill:#fee2e2,stroke:#ef4444
